TITLE
Generation of virtual memory from relatively highly self-reactive is a mechanism of self-tolerance

ORGANISM
Mus musculus

SUMMARY
Virtual memory T cell show an intermediate expression profile in between naïve and true memory T cells. Most plausible explanation is that they trigger partial memory differentiation program.

DESIGN
Analysis of naïve, virtual memory, and true memory CD8+ T cells

PLATFORM
GPL17021 Illumina HiSeq 2500 (Mus musculus)
