TITLE
Copy number variation between BJ cells and BJ-derived iPS cells

ORGANISM
Homo sapiens

SUMMARY
Copy number variation analysis between BJ cells and BJ cells derived iPS cells which were established by using non-transmissible measles virus vector

DESIGN
DNA derived from BJ cell and iPS cells were extracted and analysis using CGH array

PLATFORM
GPL10154 Agilent-021365 SurePrint G3 Human CNV Microarray 2x400K (Feature Number version)
